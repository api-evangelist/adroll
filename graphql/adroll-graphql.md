# AdRoll GraphQL API

AdRoll is a performance advertising platform for retargeting and prospecting. Their API covers campaign management, audience segments, ad creative management, attribution, and cross-channel reporting.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from AdRoll REST API
